# Rent Master - User Manual

## Introduction
Rent Master is a comprehensive property management application designed to help landlords manage properties, tenants, rent collections, and expenses efficiently.

## Getting Started

### Installation
1. Install the provided `.apk` file on your Android device.
2. Open the **RentMaster** app.

### Permissions
Upon first use, the app may request the following permissions:
- **Notifications**: To send rent due reminders.
- **SMS**: To send payment receipts or reminders (if enabled).
- **Storage**: To save generated PDF receipts and reports.

Please **Allow** these permissions for the app to function correctly.

## Key Features & How to Use

### 1. Dashboard
The home screen provides a quick summary:
- **Total Collected**: Total rent collected.
- **Total Expenses**: Total expenses recorded.
- **Quick Actions**: Add Property button (+) at the bottom.

### 2. Manage Properties & Rooms
Before adding tenants, you must set up your properties.
- **Add Property**: Tap the `+` button -> Select "Add Property". Enter Property Name and Address.
- **Add Value**: Tap on a Property to view details -> Add Rooms (e.g., "Flat 101", "Room A") and set the monthly rent amount.

### 3. Manage Tenants
- **Add Tenant**: Go to the **Tenants** tab or Property details.
- **Assign to Room**: Select a Property and Room -> Click "Add Tenant".
- **Details**: Enter Name, Phone, Email, and Move-in Date.

### 4. Record Rent & Payments
- **Record Payment**: Go to **Rent Ledger** or select a Tenant.
- **Enter Information**: Input the Amount Paid, Date, and Payment Mode (Cash/Online).
- **Receipts**: The app generates a PDF receipt automatically or upon request.

### 5. Expense Tracking
- **Add Expense**: Go to **Expenses** -> Click `+`.
- **Details**: detailed description (e.g., "Plumbing Repair"), Amount, and Date.
- **Track**: View total expenses to calculate net profit.

### 6. Reports
Navigate to the **Reports** section to generate summaries:
- **Room-wise Report**: Income per room.
- **Tenant-wise Report**: Payment history per tenant.
- **Export**: Reports can be saved as PDF files for your records.

### 7. Automatic SMS Notifications (Key Feature)
Rent Master keeps everyone in the loop with automated SMS updates:
- **For Tenants**: They receive an instant SMS receipt whenever a payment is recorded.
- **For Landlords**: You get confirmation that the payment has been logged.
- **Rent Due Reminders**: The system automatically sends text reminders to tenants when rent is due, ensuring timely payments without manual follow-up.
*Note: Standard SMS charges may apply depending on your mobile carrier.*

## Troubleshooting
- **Data Not Refreshing?** Click the "Refresh" button on the dashboard to update rent records.
- **PDF Not Saving?** Ensure Storage permission is enabled in Android Settings.
